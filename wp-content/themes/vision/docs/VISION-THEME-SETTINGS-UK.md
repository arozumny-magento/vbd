# Основні налаштування теми Vision

Документація описує, які налаштування можна змінювати в адмін-панелі WordPress для теми Vision і що саме вони роблять.

---

## Де знаходяться налаштування

У меню WordPress: **Налаштування → Vision Style Settings** (або **Settings → Vision Style Settings**).

Сторінка має вкладки:
- **General** (Загальні)
- **Blocks Style** (Стиль блоків)
- **ENG** (англійська мова)
- **UKR** (українська мова)
- **ARA** (арабська мова)

---

## 1. Вкладка «General» (Загальні)

### 1.1 Header (Шапка сайту)

| Налаштування | Що робить |
|--------------|-----------|
| **Header logo** | Логотип у шапці сайту. Кнопки «Select image» / «Remove» — вибрати зображення або прибрати. |

---

### 1.2 Main (Головне)

**Соціальні посилання**  
Поля для посилань на профілі: Instagram, YouTube, Facebook, LinkedIn. Введіть повний URL (наприклад, `https://www.instagram.com/...`).

**Кольори сайту**

| Налаштування | Опис |
|--------------|------|
| **Main website color** | Основний колір сайту (наприклад, темно-синій для тексту/акцентів). |
| **Accent color** | Акцентний колір (наприклад, золотистий) — посилання, кнопки, підсвітка. |

---

### 1.3 Footer (Підвал)

| Налаштування | Що робить |
|--------------|-----------|
| **Footer logo** | Логотип у підвалі. Вибрати зображення або прибрати. |
| **Contact form shortcode** | Шорткод форми зворотного зв’язку (наприклад, Contact Form 7). Формат: `[contact-form-7 id="123" title="Contact"]`. |
| **Contact form background color** | Колір фону блоку з формою контакту в підвалі. |
| **Contact form image** | Фонове зображення для блоку контакту (ліва частина). |

---

### 1.4 Block theme presets (Пресети блоків)

Кожен контентний блок на сторінках може мати один із пресетів: **White**, **Light** або **Dark**. Тут задаються кольори цих пресетів.

| Пресет | Налаштування | Що це |
|--------|--------------|--------|
| **White preset** | Block background, Text | Фон і колір тексту для блоків у стилі «білий». |
| **Light preset** | Block background, Text | Фон і колір тексту для блоків у стилі «світлий». |
| **Dark preset** | Block background, Text | Фон і колір тексту для блоків у стилі «темний». |

Ці кольори використовуються на головній та інших сторінках у блоках з класами `block-style-white`, `block-style-light`, `block-style-dark`. Блок з типом **Custom** має власні кольори, задані в ACF.

---

### 1.5 Block hover effect (Ефект при наведенні на блок)

Налаштування впливають на те, як виглядають блоки при наведенні курсора.

| Налаштування | Що робить |
|--------------|-----------|
| **Enable hover effect** | Увімкнути/вимкнути ефект при наведенні. Якщо вимкнено — блоки не змінюють вигляд при наведенні. |
| **On hover effect for block** | Тип ефекту: None, Color fade, Gradient shift, Slide line, Glow, Shine sweep, Fill from left, Subtle lighten, Scale up, Fade invert, Soft highlight. |
| **On hover text color** | Колір тексту в блоці при наведенні (наприклад, золотистий). |
| **On hover background color** | Колір фону блоку при наведенні. Можна вказати `transparent`, щоб фон не змінювався. |

Ефект застосовується до блоків зі стилями: **white**, **light**, **dark** та **custom**. Нижче на сторінці налаштувань є прев’ю — можна навести курсор на зразки блоків і побачити обраний ефект.

---

## 2. Вкладка «Blocks Style»

Повторює налаштування **Block theme presets** та **Block hover effect** з вкладки General: пресети кольорів (White, Light, Dark) і всі параметри ефекту при наведенні на блок. Зручно мати їх разом у одному місці для стилів блоків.

---

## 3. Вкладки мов: ENG, UKR, ARA

Для кожної мови (англійська, українська, арабська) налаштовуються **шрифти**: розмір, міжрядковий інтервал (line height) та насиченість (weight).

Є два підрежими:
- **Desktop** — параметри для комп’ютерів і планшетів;
- **Mobile** — параметри для мобільних пристроїв (можна задати менші розміри тощо).

### Що налаштовується

| Елемент | Параметри | Де застосовується |
|---------|-----------|-------------------|
| **Main font** | Size, Line height, Weight | Основний текст сторінки. |
| **Block Heading font** | Size, Line height, Weight | Заголовки всередині блоків (елементи з класом .vision-block-heading). |
| **H1 … H6** | Size, Line height, Weight | Заголовки 1–6 рівня по всьому сайту. |

**Розмір (Size)** вводиться в rem (наприклад: 1, 1.25, 1.5, 2).  
**Weight** — число від 100 до 900 (400 — normal, 600 — semibold, 700 — bold).

---

## Короткий чеклист

- **Логотипи:** General → Header logo, Footer logo.  
- **Соцмережі та кольори сайту:** General → Main (social links, Main website color, Accent color).  
- **Кольори блоків (white/light/dark):** General або Blocks Style → Block theme presets.  
- **Ефект при наведенні на блок:** General або Blocks Style → Block hover effect (увімкнути, тип ефекту, колір тексту та фону при наведенні).  
- **Форма контакту в підвалі:** General → Footer (shortcode, колір фону, зображення).  
- **Шрифти для кожної мови:** вкладки ENG, UKR, ARA (Desktop / Mobile, Main font, Block Heading, H1–H6).

Після змін обов’язково натисніть **Save Changes** внизу сторінки.
