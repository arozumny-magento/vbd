# Налаштування головної сторінки (Vision) — розширена версія

Документація описує всі блоки ACF групи **«Home Page New»** для головної сторінки та детально — кожне поле й його призначення.

У групі **6 основних блоків**: Hero Block, About Section, Services Section, Testimonials Section, Partners Section, News Section.

---

## Як відкрити налаштування

1. У меню WordPress: **Сторінки** → вибрати сторінку, яка є **головною** (Home).
2. Переконатися, що для сторінки обрано шаблон **«VBD Home»** (у блоці або в мета-полі шаблону).
3. На сторінці редагування з’явиться група полів **«Home Page New»** з усіма блоками нижче.

---

# Блок 1. Hero Block (Герой)

Перший екран головної сторінки: заголовок і фонове відео.

| Поле | Тип | Обов’язкове | Опис |
|------|-----|-------------|------|
| **Main Title** | Текст | Так | Головний слоган (H1). Відображається над відео. |
| **Hero Image (Optional)** | Зображення | Ні | Постер для відео (показується до завантаження або якщо відео не відтворюється). Формат: URL. |
| **Hero Video** | Файл | Так | Відео для фону. Рекомендовано MP4. Формат повернення: URL. |

**Порада:** Відео краще стиснути для вебу, щоб швидко завантажувалось. Постер варто мати того ж співвідношення сторін, що й відео.

---

# Блок 2. About Section (Секція «Про нас»)

Один ряд із двома колонками (лівий і правий блок). Структура рядів та сама, що в інших секціях з рядами (див. спільний опис блоків нижче).

| Поле | Тип | Опис |
|------|-----|------|
| **Section Header** | Текст | Заголовок секції (наприклад, «About» або «Про нас»). Відображається над рядом. |
| **Section Row** | Група (не повторювач) | Один ряд. Всередині — **Left Block** і **Right Block**. |

**Left Block / Right Block:** кожен блок налаштовується через Block Settings, тип (Text або Media), тему, посилання та контент (заголовок, текст, зображення) — див. розділ **«Спільна структура блоків у ряді»** нижче.

---

# Блок 3. Services Section (Секція «Послуги»)

Секція з багатьма рядами; у кожному ряді — лівий і правий блок (текст або медіа).

| Поле | Тип | Опис |
|------|-----|------|
| **Section Header** | Текст | Заголовок секції (наприклад, «Services»). |
| **Section Row** | **Повторювач** | Кнопка **«Add Row»** додає новий ряд. У кожному ряді — **Left Block** і **Right Block**. Мінімум/максимум рядів не обмежені. |

Кожен **Left Block** і **Right Block** має повну структуру: Block Type (Text/Media), Block Theme, Block Link, Text Block або Media Block, посилання — див. **«Спільна структура блоків у ряді»**.

---

# Блок 4. Testimonials Section (Відгуки)

Секція відгуків. Кожен ряд — це один або два відгуки; контент береться з записів типу **Testimonial** (обраних через зв’язок).

| Поле | Тип | Опис |
|------|-----|------|
| **Section Header** | Текст | Заголовок секції (наприклад, «Testimonials»). |
| **Section Row** | **Повторювач** | Кнопка **«Add Row»**. У кожному ряді — **Left Block** і **Right Block**. |

**Особливість секції Testimonials:** у Left Block і Right Block **немає** вибору типу Text/Media і ручного введення тексту. Замість цього:

| Поле (всередині Left Block / Right Block) | Тип | Опис |
|-------------------------------------------|-----|------|
| **Block Settings** | Група | **Block Theme** (Light / Dark / White / Custom), **Order** (число, для порядку). |
| **Custom Styles** | Група | З’являється, якщо обрано **Custom**. Поля: **Background Color**, **Text Color**. |
| **Choose Testimonial** | Зв’язок (Relationship) | Вибір **одного** запису з типу записів **Testimonial**. Контент блоку (компанія, автор, текст відгуку, лого) підтягується з обраного запису. |

Тобто для кожного блоку в ряді ви обираєте готовий відгук з сайту; тема блоку (Light/Dark/White/Custom) і кольори для Custom задаються тут же.

---

# Блок 5. Partners Section (Партнери)

Секція з заголовком, кольором фону та списком логотипів партнерів.

| Поле | Тип | Опис |
|------|-----|------|
| **Section Header** | Текст | Заголовок (наприклад, «Partners»). |
| **Background Color** | Колір (Color Picker) | Колір фону всієї секції. За замовчуванням — #00012E. |
| **Partners Logo** | **Повторювач** | Список партнерів. Кнопка **«Add Row»** додає один пункт. |

**Поля всередині одного рядка повторювача «Partners Logo»:**

| Поле | Тип | Опис |
|------|-----|------|
| **Title** | Текст | Назва партнера (для підказки або доступності). |
| **Link** | URL | Посилання на сайт партнера. |
| **Logo** | Зображення | Логотип. Формат повернення: URL. |

На сайті зазвичай відображається сітка логотипів; клік по лого веде на вказане посилання.

---

# Блок 6. News Section (Новини)

Секція з новинами у вигляді рядів; у кожному ряді — лівий і правий блок (наприклад, зображення + текст з посиланням).

| Поле | Тип | Опис |
|------|-----|------|
| **Section Header** | Текст | Заголовок секції (наприклад, «News»). |
| **Section Row** | **Повторювач** | Кнопка **«Add Row»**. У кожному ряді — **Left Block** і **Right Block**. |

Left Block і Right Block мають ту саму структуру, що в About та Services: Block Type (Text/Media), Block Theme, Block Link, Text Block або Media Block, посилання — див. **«Спільна структура блоків у ряді»**.

---

# Спільна структура блоків у ряді (Left Block / Right Block)

Використовується в секціях: **About**, **Services**, **News**. (У **Testimonials** структура інша — там лише Block Theme, Custom Styles і Choose Testimonial.)

## Крок 1. Block Settings (Налаштування блоку)

| Поле | Варіанти | Опис |
|------|----------|------|
| **Block Type** | **Text** / **Media** | **Text** — блок із заголовком, текстом і опційно «Read More». **Media** — блок тільки з фоновим зображенням (на весь блок). |
| **Block Theme** | **Light** / **Dark** / **White** / **Custom** | Готові кольорові пресети з налаштувань теми або власні кольори. |
| **Block Link** | **Page** / **URL** | Куди веде клік по всьому блоку: вибір сторінки з сайту або довільний URL. |
| **Order** | Число | Порядковий номер (для сортування, якщо потрібно). |

## Крок 2. Custom Styles (тільки якщо Block Theme = Custom)

З’являються після вибору **Custom** у Block Theme.

| Поле | Опис |
|------|------|
| **Background Color** | Колір фону блоку (наприклад, #FFFFFF). |
| **Text Color** | Колір тексту в блоці (наприклад, #000000). |

Ці кольори перевизначають пресети Light/Dark/White лише для цього блоку.

## Крок 3а. Якщо Block Type = Text — Text Block

| Поле | Тип | Опис |
|------|-----|------|
| **Header** | Текст | Заголовок блоку (не обов’язково). |
| **Text** | Редактор (WYSIWYG) | Основний текст. Підтримуються списки, жирний, посилання тощо. |
| **Read More** | Так/Ні (True/False) | Показувати чи ні текст «Read More» (зазвичай внизу блоку). |

Потім обов’язково заповнити посилання для кліку по блоку (див. крок 4).

## Крок 3б. Якщо Block Type = Media — Media Block

| Поле | Тип | Опис |
|------|-----|------|
| **Background Image** | Зображення | Зображення на весь блок (cover). Формат повернення: URL. |

Посилання для кліку по блоку — у крокі 4.

## Крок 4. Посилання для всього блоку

Залежить від **Block Link** у Block Settings:

- Якщо обрано **Page**: заповнити поле **Link to Page** — вибір однієї сторінки з сайту (Relationship, повертає об’єкт).
- Якщо обрано **URL**: заповнити поле **Block URL** — довільне посилання (тип URL).

Якщо посилання не задане, блок може відображатися без кліку або з неактивною кнопкою «Read More» (залежить від верстки).

# Підсумкова таблиця по секціях

| № | Блок | Заголовок секції | Рядки | Особливості |
|---|------|------------------|-------|-------------|
| 1 | Hero Block | — | Один екран | Main Title, Hero Image, Hero Video. |
| 2 | About Section | Section Header | Один ряд (група) | Left Block + Right Block (Text/Media, тема, посилання). |
| 3 | Services Section | Section Header | Повторювач рядів | У кожному ряді Left + Right Block (Text/Media). |
| 4 | Testimonials Section | Section Header | Повторювач рядів | Left + Right Block: лише Block Theme, Custom Styles, **Choose Testimonial** (зв’язок з записом Testimonial). |
| 5 | Partners Section | Section Header | — | Background Color + повторювач Partners Logo (Title, Link, Logo). |
| 6 | News Section | Section Header | Повторювач рядів | У кожному ряді Left + Right Block (Text/Media). |

# Поширені сценарії

- **Тільки текст у блоці:** Block Type = Text, заповнити Header і Text, при потребі увімкнути Read More і задати Block Link (Page або URL).
- **Тільки картинка на весь блок:** Block Type = Media, завантажити Background Image, при потребі задати Link to Page або Block URL.
- **Відгук у секції Testimonials:** у ряді Section Row обрати Left Block і Right Block; у кожному — Block Theme і **Choose Testimonial** (вибрати запис з типу Testimonial).
- **Партнер:** у Partners Section натиснути «Add Row» в Partners Logo, вказати Title, Link, Logo.
- **Новина в два блоки:** у News Section додати ряд; лівий блок — Media (зображення новини) + посилання на запис; правий блок — Text (заголовок, уривок, Read More) + посилання на ту саму новину.

Після будь-яких змін обов’язково натисніть **«Оновити»** або **«Опублікувати»** сторінку.
